# Jumporia
- A 2D platformer where you conquer 5 exciting levels. Reach the finish line to go to the next level!
- Where the primary challenge is navigating a character through levels filled with platforms, obstacles, and enemies. Players must jump between platforms, deafeat enemies, and collect candles to finsh the current level.
- There is the option of changing the sound setting in the main menu.

## How To Run The Game?
- To run the game you should open the itch.io link, then click the button "Run Game". So simple.

## How To Play The Game?
- To play the game correctly, you should click the "Start Game" button in the main menu after running the game. Then select the level that you want to play.
- To finish the level, you should complete each level objectives, once you finish them, a key will appear next to the player, collect it.
- Once you collect the key, search for the finish line and go into it, then choose if you want to go to the next level or return to the main menu.

## Controllers.
- Movement is controlled using A and D or the ← and → arrow keys to move the character horizontally across the level.
- The player can jump using W, the ↑ arrow key, or the spacebar.
- Attacks are performed using the LMB or the Ctrl key.
